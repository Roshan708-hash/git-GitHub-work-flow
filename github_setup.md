\# GitHub Setup Guide (Beginner Friendly)



This guide explains:

1\. How to create a GitHub account

2\. How to connect Git on your computer

3\. How to add an SSH key (safe login without password)



No prior knowledge required.



---



\## What is GitHub?

\- GitHub is a website to \*\*store your code online\*\*

\- It helps you \*\*backup\*\*, \*\*share\*\*, and \*\*collaborate\*\*



Think of GitHub like \*\*Google Drive for code\*\* ☁️



---



\## 1. Create a GitHub Account (Signup)



1\. Open browser

2\. Go to: https://github.com

3\. Click \*\*Sign up\*\*

4\. Login to GitHub



✅ GitHub account is ready
